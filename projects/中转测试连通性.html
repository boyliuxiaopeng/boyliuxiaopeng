GPT / Claude 中转站连通性测试

本页面只在你的浏览器本地运行。它会对中转站发起 /v1/models 和最小聊天请求，用于判断：域名是否可达、鉴权是否通过、模型是否可调用、响应延迟是多少。

注意：浏览器直连第三方 API 会受 CORS 限制。若结果显示 Failed to fetch / CORS，不一定代表中转站不可用；请使用页面生成的 curl 命令在终端复测。API Key 会从浏览器直接发送到你填写的 Base URL，不会发送给其它地方。
默认不会保存 API Key。

结果解读：
1. HTTP 200 且返回内容中有模型回复，说明中转站、Key、模型名基本可用。
2. 401/403 多数是 Key、鉴权头或额度权限问题。
3. 404 多数是 Base URL 或协议类型不匹配，例如把 Claude 原生站点按 OpenAI 兼容协议测了。
4. 429 是限流或余额/并发限制。
5. Failed to fetch 常见于 CORS、DNS、TLS、代理拦截或网络不可达，建议复制 curl 到终端复测。
